HTML SEMANTIC TAGS CHEAT SHEET


Semantic is an important thing to make a better version of our code
and become a great web developer.
In the table below, you can find some semantic tags.



TAG DESCRIPTION
<header> Used for either navigational links, introductory content or headings.
<main> Used to encapsulate the dominant content within a webpage.
<footer> Contains informations such as contact informations, copyright informations, terms of use, site map,
reference to the top of page links.
<nav> Used to define a block of navigation links such as menus and tables of content, way to label our
navigations links.
<span> Contains short pieces of text, they are used to separate small pieces of content that are on the
same line as other content
<embed> Self-closing tag, it can embed any media content including videos, audio files, gifs, from an
external source.
<section> Defines elements in a document, such as chapters, headings, or any other area of the document
with the same theme.
<aside> Used to mark additional informations that can enhance another element but isn't required in order
to understand the main content. Common uses are for bibliographies, endnotes, pull quotes,
editorial sidebars, additional informations.
<figure> Used to encapsulate media such as an image, an illustration, a diagram, a code snippet.
<figcaption> Used to add a caption to the imae, or describe the media in the <figure> tag where he will usually
go.
<article> Hold content such as articles, blog, comments, magazines.
